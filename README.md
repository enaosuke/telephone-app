# おでんわアプリ 📞

幼児向けの電話ごっこアプリです。iPhoneの電話画面のようなデザインで、ボタンを押すと音が鳴ります。

## 機能

- **大きな数字ボタン** … 幼児でも押しやすいサイズ
- **押すと音が鳴る** … 各ボタンでDTMF風の音（Web Audio API）
- **電話をかけるボタン** … 緑のボタンで「かけちゅう…」表示
- **PWA対応** … ホーム画面に追加してアプリのように利用可能

## 技術

- React 18 + Vite 6
- vite-plugin-pwa（オフライン対応・インストール可能）
- Web Audio API（ボタン音）
