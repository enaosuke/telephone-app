# おでんわアプリ 📞

幼児向けの電話ごっこアプリです。iPhoneの電話画面のようなデザインで、ボタンを押すと音が鳴ります。

## URL
https://enaosuke.github.io/telephone-app/

## 機能

- **大きな数字ボタン** … 幼児でも押しやすいサイズ
- **押すと音が鳴る** … 各ボタンでDTMF風の音（Web Audio API）
- **電話をかけるボタン** … 緑のボタンで発信。発信中は番号の上に「発信中…」を表示し、赤い通話終了ボタンで切断音を鳴らして終了
- **PWA対応** … ホーム画面に追加してアプリのように利用可能
- **隠しギミック** … 「110」と入力して発信すると…？

## 技術

- React 18 + Vite 6
- vite-plugin-pwa（オフライン対応・インストール可能）
- Web Audio API（ボタン音）

## 音素材

発信音・切断音・効果音などは [効果音ラボ](https://soundeffect-lab.info/) の素材を使用しています。
